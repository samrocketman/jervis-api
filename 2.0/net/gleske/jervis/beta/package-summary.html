Package net.gleske.jervis.beta Description

Contains unreleased APIs in development. These APIs have little to no testing written. The feature is in incubation and subject to rapid change. Avoid relying on any APIs in this package. Changes made in these APIs have no impact on a major or minor release. A minor feature release could have major API breaking changes in this package. You have been warned.
